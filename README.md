# Quant Risk Analysis using Monte Carlo Simulation

## Overview
This project uses Monte Carlo simulation and Geometric Brownian Motion to analyze stock price behavior and estimate future risks.

## Features
- Historical data analysis
- Log return calculation
- Volatility estimation
- Monte Carlo simulation (2000+ scenarios)
- Price distribution visualization
- Confidence interval analysis

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- yFinance

## Output
- Simulated future price paths
- Probability distribution
- Expected price trend with risk bands

## Future Scope
- Value at Risk (VaR)
- Probability of profit
- Portfolio optimization
- AI-based recommendations
